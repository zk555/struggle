项目介绍

  • 面向学习型的开源框架，简洁高效，减少过渡封装，展现技术本质
  • Springboot作为基础框架，使用mybatis作为持久层框架
  • 使用官方推荐的thymeleaf做为模板引擎，shiro作为安全框架,主流技术，“一网打尽”
  • 极简配置，一键前后台代码生成

   涉及技术

  • SpringBoot ：项目基础框架
  • mybatis ：项目基础框架
  • SpringMvc ：项目基础框架
  • Redis : 保存用户登录session
  • Maven : 项目管理工具
  • netty : 完成rpc调用网关
  • zookeeper : rpc注册地址
  • rabbitMq : 研究使用，不涉及到该项目
  • kafka : 研究使用，不涉及到该项目
  • shiro : 用户登录后安全管控
  • Aop : 通过自定义日志拦截器汇总所有请求
  • Druid : 数据源监控
  • Quartz : 定时器引入，定时向web端推送消息（webService）
